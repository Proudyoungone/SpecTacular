Place your model reference images in this folder.

Recommended naming:
- rayban_meta_gen2.png
- rayban_meta_gen1.png
- meta_rayban_display.png
- oakley_meta.png

Guidelines:
- Use images you created yourself, product images you have permission to use, or manufacturer/retailer images whose license allows reuse.
- Keep them reasonably cropped so the glasses are clearly visible.
- JPG or PNG is fine.
- Around 900px wide is usually enough for the report.

If an expected file is missing, SpecTacular will still show the detected model and capabilities without an image.
